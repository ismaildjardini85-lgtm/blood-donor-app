# 🌐 نشر تطبيق إدارة متبرعي الدم على الإنترنت

## الخيار 1: النشر على Netlify (الأسهل والأسرع) ⭐

### الخطوات:

1. **إنشء حساب على Netlify**
   - اذهب إلى https://netlify.com
   - اضغط "Sign Up"
   - اختر "GitHub" أو "Google"

2. **تحضير الملفات**
   - تأكد من أن لديك:
     - `index.html`
     - `renderer.js`
     - `package.json`

3. **رفع الملفات**
   - في Netlify، اختر "Drag and drop to deploy"
   - اسحب مجلد `extracted_app` أو اختر الملفات الثلاثة فقط
   - انتظر الرفع (دقيقة واحدة تقريباً)

4. **الحصول على الرابط**
   - بعد النشر، ستحصل على رابط مثل: `https://xxx-xxx.netlify.app`
   - شارك هذا الرابط مع زملائك!

---

## الخيار 2: النشر على Vercel

### الخطوات:

1. **إنشء حساب على Vercel**
   - اذهب إلى https://vercel.com
   - اضغط "Sign Up"

2. **رفع المشروع**
   - اختر "Import Project"
   - أو "Drag and drop"

3. **الانتظار**
   - سيتم نشر التطبيق تلقائياً

---

## الخيار 3: النشر على GitHub Pages (مجاني ومجتمعي)

### الخطوات:

1. **إنشء مستودع على GitHub**
   - اذهب إلى https://github.com/new
   - سمِّ المستودع: `blood-donor-management`
   - اختر Public

2. **رفع الملفات**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/blood-donor-management.git
   git push -u origin main
   ```

3. **تفعيل Pages**
   - اذهب إلى Settings → Pages
   - اختر Branch: main
   - انتظر النشر

4. **رابطك**
   - `https://YOUR_USERNAME.github.io/blood-donor-management`
