# 🚀 النشر السريع على Netlify (أسهل طريقة)

## في 3 دقائق فقط!

### الخطوة 1: جهّز الملفات
```
extracted_app/
├── index.html
├── renderer.js
└── package.json (اختياري)
```

### الخطوة 2: اذهب إلى Netlify
- افتح: https://netlify.com
- اضغط: **Sign Up** (أو تسجيل الدخول)

### الخطوة 3: النشر
**الطريقة الأولى (الأسهل):**
1. في الصفحة الرئيسية، ستجد "Drag files to deploy"
2. اسحب مجلد `extracted_app` وأفلته هناك
3. انتظر الانتهاء (30 ثانية تقريباً)

**الطريقة الثانية:**
1. اضغط "New site" → "Deploy manually"
2. اختر الملفات الثلاثة (index.html, renderer.js, package.json)
3. ارفعها

### الخطوة 4: احصل على الرابط
بعد النشر مباشرة، ستحصل على رابط مثل:
```
https://xxx-xxxxxx.netlify.app
```

### الخطوة 5: شارك الرابط
أرسل الرابط لزملائك:
```
https://xxx-xxxxxx.netlify.app
```

## هل تريد رابط أفضل؟

بعد النشر على Netlify:
1. اضغط على إعدادات الموقع
2. اذهب إلى "Site Settings"
3. غيّر اسم الموقع إلى شيء مثل:
   - `blood-donor-app`
   - `donations-tracker`

سيصبح الرابط: `https://blood-donor-app.netlify.app`
