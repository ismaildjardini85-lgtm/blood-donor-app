## 🚀 تشغيل التطبيق مع JSON Server

### الطريقة الأولى - تشغيل الخادم تلقائياً:

```bash
npm run server
```

### الطريقة الثانية - تشغيل يدوي:

```bash
json-server --watch db.json --port 3000
```

---

### بعد تشغيل الخادم:

1. افتح `index.html` في المتصفح
2. أضف بيانات المتبرعين
3. البيانات ستُحفظ في `db.json` مباشرة

---

### الآن يمكنك:
- ✅ فتح التطبيق من أجهزة مختلفة (نفس الشبكة)
- ✅ جميع الأجهزة ترى نفس البيانات تلقائياً
- ✅ البيانات محفوظة دائماً في `db.json`
