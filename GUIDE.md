# 📋 دليل استخدام التطبيق

## الميزات الأساسية:

### 1️⃣ إضافة متبرع جديد
- ملأ النموذج بالبيانات الكاملة
- اضغط "Ajouter Donneur"
- سيتم حفظ البيانات تلقائياً

### 2️⃣ البحث والتصفية
- اكتب في حقل "Rechercher" 
- يتم البحث عن: الاسم، الهاتف، رقم الكيس، الفئة، التاريخ

### 3️⃣ حذف متبرع
- اضغط زر "Supprimer" في الصف
- تأكد قبل الحذف

### 4️⃣ الإحصائيات 📊
- اضغط زر "إحصائيات"
- ستجد:
  - عدد الأكياس في كل سنة
  - إجمالي لكل فئة دم
  - تفاصيل شهرية لكل فئة

### 5️⃣ تصدير البيانات 📥
- **تصدير CSV**: لـ Excel أو Sheets
- **تصدير JSON**: للنسخ الاحتياطي

### 6️⃣ استيراد البيانات 📤
- اختر ملف JSON
- اختر: استبدال أو إضافة
- سيتم تحديث البيانات

### 7️⃣ حذف جميع البيانات 🗑️
- ⚠️ هذا الإجراء لا يمكن التراجع عنه
- سيطلب تأكيد منك

---

## نصائح مفيدة:

💾 **النسخ الاحتياطية**: 
- قم بتصدير JSON بشكل دوري
- احفظها في مكان آمن

🔄 **مشاركة البيانات مع الزملاء**:
- صدّر البيانات
- أرسل الملف للزملاء
- يستطيعون استيرادها

📱 **التطبيق يعمل على**:
- الكمبيوتر
- الجوال
- اللوحة

A+, A-, B+, B-, AB+, AB-, O+, O-

## تنبيهات:

⚠️ عند حذف بيانات المتصفح، ستُفقد البيانات  
⚠️ استخدم النسخ الاحتياطية بانتظام
